# Selector Prysmian Ecuador

Herramienta web estática para selección de cables Prysmian.

## Funcionalidades

- Mapa de decisión interactivo
- Navegación por categorías
- Visualización de información del cable
- Compatible con GitHub Pages

## Publicación

Subir todos los archivos a un repositorio GitHub.

Activar:

Settings → Pages → Deploy from Branch

Seleccionar:

main / root

La aplicación quedará disponible en:

https://usuario.github.io/prysmian-selector/
